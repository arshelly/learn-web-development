Introduction to html

  • html stands for hypertext markup language
  • html is the standard markup language for creating webpages
  • html describes the structure of the webpage
  • html consist of a series of the webpages
  • html element tell the browser how to display the content
  • html element label pieces of the content such as"this is a heading " , "this is a pragraph" , "this is a link" ,etc.

what is an html element

An HTML element is a type of HTML(HyperTextMarkupLanguage)document component, one of the several types of HTML nodes(there are also text nodes,comment nodes.HTML elements are those which adds semantics and formatting to theparts of document.each element in html have its attributes specified. Example:divspan,link,title etc

HTML tags are like keywordswhich defines that how web browser will format and displaythe content.With the use of tags one can distinguish between HTML content and a simple content.
